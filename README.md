## 概况

- 自己平时用的presentation
- 使用[nodeppt](https://github.com/ksky521/nodeppt)制作
- 可用主题 colors，moon，blue，dark，green，light

## 运行

```
npm install -g noteppt
noteppt start
```

## 生成

```
nodeppt generate file.md  public/path -a
```
