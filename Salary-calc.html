工资与五险一金计算

关鹏

公式

  • 核心公式
    • 实发工资 = 应发工资 － 个人五险一金 － 个人所得税
    • 个人资产 = 实发工资 + 个人五险一金 + 公司五险一金
  • 解释
    • 应发工资 = 公司报价（比如20K/月，25K/月）
    • 个人五险一金 = 个人按个人工资比例缴纳的保险
    • 公司五险一金 = 公司按个人工资比例缴纳的保险
    • 个人所得税 = (应发工资 － 个人五险一金)超过3500元后按比例缴纳的税收

五险一金

  • 不同城市五险一金缴费比例、最高缴费标准有差别，假设为以下比例
    • 个人缴费23% = 养老8% + 医疗2% + 失业1% + 公积金12%
    • 公司缴费43.5% = 养老20% + 医疗8% + 失业 2% + 生育1% + 工伤0.5% + 公积金12%
  • 举例计算：
    • 应发工资 = 10000
    • 个人五险一金 = 10000 * 23% = 2300
    • 公司五险一金 = 10000 * 43.5% = 4350

实发工资 = 10000(应发工资) － 2300(个人五险一金) － 个人所得税

个人资产 = 实发工资 + 2300(个人五险一金) + 4350(公司五险一金)

个人所得税计算

  • 举例计算
    • 应发工资 = 10000
    • 个人五险一金 = 10000 * 23% = 2300
    • 应纳税所得额 = (应发工资 － 个人五险一金) － 3500 = (10000 － 2300) － 3500 = 4200
    • 个人所得税 = 应纳税所得额 × 税率 － 速算扣除数 = 4200 × 10% － 105 = 315（等同于分段计算，速算扣除数含义）

7385(实发工资) = 10000(应发工资) － 2300(个人五险一金) － 315(个人所得税)

14035(个人资产) = 7385(实发工资) + 2300(个人五险一金) + 4350(公司五险一金)

各阶段月薪情况

7385(实发工资) = 10000(应发工资) － 2300(个人五险一金) － 315(个人所得税)

14350(公司总支出) = 10000(应发工资) + 4350(公司五险一金)

结论

  • 工资较低的时候，到手工资比例主要受五险一金影响
  • 随着工资的升高，五险一金保持不变，到手比例主要受超额累进税率的影响
  • 工资趋于无穷大，到手工资比例将无限接近55%，因为工资个税最高边际税率为45%
  • 如果公司减少公司五险一金支出，那么亏本的是个人
